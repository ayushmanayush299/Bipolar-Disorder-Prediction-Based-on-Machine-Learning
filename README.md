# Bipolar-Disorder-Prediction-Based-on-Machine-Learning

End-to-end Machine Learning project for Bipolar Disorder Prediction using structured behavioral and clinical data. Implements preprocessing, feature engineering, and classification models for risk estimation, integrated with a Flask web application for real-time predictions and interactive user input.
---

## 📌 Overview

This project is an end-to-end Machine Learning application designed to predict the likelihood of Bipolar Disorder using structured behavioral and clinical data. It integrates data preprocessing, model training, and deployment through a web-based interface for real-time predictions.

---

## 🚀 Features

* 🔍 Machine Learning-based prediction system
* 📊 Data preprocessing and feature engineering
* 📈 Risk estimation based on user input
* 🌐 Interactive web interface using Flask
* ⚡ Real-time prediction output

---

## 🛠 Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Framework:** Flask

## ▶️ How to Run the Project

### Step 1: Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/bipolar-disorder-prediction-ml.git
cd bipolar-disorder-prediction-ml
```

### Step 2: Install dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run the application

```bash
python app.py
```

### Step 4: Open in browser

http://127.0.0.1:5000/
