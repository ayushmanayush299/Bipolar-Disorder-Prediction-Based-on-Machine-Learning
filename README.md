# Bipolar-Disorder-Prediction-Based-on-Machine-Learning

End-to-end Machine Learning project for Bipolar Disorder Prediction using structured behavioral and clinical data. Implements preprocessing, feature engineering, and classification models for risk estimation, integrated with a Flask web application for real-time predictions and interactive user input.


##  Overview

This project is an end-to-end Machine Learning application designed to predict the likelihood of Bipolar Disorder using structured behavioral and clinical data. It integrates data preprocessing, model training, and deployment through a web-based interface for real-time predictions.Its Basically a research based project.


##  Features

* Machine Learning-based prediction system
* Data preprocessing and feature engineering
* Risk estimation based on user input
* Interactive web interface using Flask
* Real-time prediction output



##  Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn ,Matplotlib, Seaborn
* **Machine Learning:** XGBoost, Random Forest, and Logistic Regression
* **Framework:** Flask
* **Frontend:** HTML, CSS
* **Tools:** Jupyter Notebook
